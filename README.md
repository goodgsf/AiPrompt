# AI提示词模板库

收集各类AI模型的提示词模板。

## 目录结构

```
ai提示词/
├── 编码生成/       # 编程相关提示词
├── 图片生成/       # 图片生成提示词（JSON + MD格式）
├── 视频生成/       # 视频生成提示词
└── 文章生成/       # 文章写作提示词
```

## 分类说明

### 编码生成
包含提升AI编程输出质量的提示词技巧：
- 结构化思考协议
- 链式验证（CoVe）
- 少样本学习
- 迭代精进循环
- 多视角提示

### 图片生成
适配 Seedream、Nano Banana Pro、Midjourney 等图片生成模型：
- 城市视觉海报
- 信息卡片设计
- 产品渲染图
- 节日海报

### 视频生成
适配 Kling、Veo、Seedance 等视频生成模型：
- 首尾帧视频转场提示词

### 文章生成
各类型文章写作提示词：
- 育儿专栏
- 通用写作模板

## 使用方式

每个模板文件包含：
- **适用模型**：标注支持的AI模型
- **提示词**：可直接使用的提示词内容
- **使用方式**：具体使用说明

将方括号 `[...]` 内的内容替换为实际需求即可使用。

## 项目说明

此项目仅提供提示词模板，不进行具体需求实现。
